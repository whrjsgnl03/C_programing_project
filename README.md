# 학생 성적 관리 프로그램 (Student Grade Manager)

이 프로그램은 학생의 과목별 성적을 입력받아 과목별 등급, 평균 점수 및 평균 등급을 계산하고 시각화된 표 형태로 출력해 주는 C언어 기반 콘솔 애플리케이션입니다.

## 주요 기능

1. **과목 및 학생 정보 입력**: 입력 가능한 최대 과목 수 및 학생 수 범위 내에서 유동적으로 조절 가능합니다.
2. **입력 값 검증**: 점수(0~100) 및 범위 밖의 입력을 방증하여 에러 메시지를 표시하고 안전한 입력을 보장합니다.
3. **등급 자동 계산**: 점수에 비례하여 등급(A~E)을 자동 산출합니다.
4. **구조체 포인터 활용**: 구조체 멤버 접근 시 포인터를 활용하여 메모리 접근을 효율화했습니다.
5. **표(Table) 형식의 결과 출력**: 정렬된 형식으로 결과를 한눈에 확인할 수 있습니다.

## 빌드 및 실행 방법

### 요구 사항
- GCC 컴파일러
- Make 빌드 도구

### 빌드하기
프로젝트 디렉토리에서 아래 명령어를 실행합니다.
```bash
make
```

### 실행하기
빌드가 완료되면 생성된 실행 파일을 실행합니다.
```bash
./student_grade
```

### 정리하기
생성된 실행 파일을 삭제하려면 아래 명령어를 실행합니다.
```bash
make clean
```
